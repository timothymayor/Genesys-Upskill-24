What distinguishes normal HTML elements from semantic HTML elements?

Normal HTML elements are those that are used to define the structure and appearance of a web page without conveying any specific meaning or purpose.
Examples of normal HTML elements include

  • div
  • span
  • b
Semantic HTML elements on the other hand, are specifically designed to convey meaning to both the browser and the developer.
These elements provide information about the content they wrap, making the HTML more descriptive and easier to understand for both humans and machines.
Examples of semantic HTML elements include
  • header
  • footer
  • section
  • article
  • nav
  • main
  • aside
  • figure
  • figcaption
By using semantic HTML elements, developers can improve the accessibility, SEO, and maintainability of their web pages.
